SSH

SSH

名称：ssh

Vault SSH 机密引擎为通过 SSH 协议访问机器提供安全的身份验证和授权功能。Vault SSH 机密引擎有助于管理对机器基础设施的访问，提供多种颁发 SSH 凭据的方法。

Vault SSH 机密引擎支持以下模式：

  • 签名的 SSH 证书
  • 一次性 SSH 密码

阅读上述内容需要了解 SSH 协议的基础知识。
